# Task 9: Configurar GitHub Actions CI/CD

## Descripción

Configurar pipeline de CI/CD con GitHub Actions para ejecutar tests automáticamente.

## Archivos a crear

1. `.github/workflows/test.yml`

## Implementación

```yaml
name: Tests

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v4
    - name: Set up JDK 17
      uses: actions/setup-java@v4
      with:
        java-version: '17'
        distribution: 'temurin'
        cache: gradle
    - name: Run Unit Tests
      run: ./gradlew testDebugUnitTest
    - name: Upload Test Results
      uses: actions/upload-artifact@v4
      if: always()
      with:
        name: test-results
        path: app/build/reports/tests/

  build:
    runs-on: ubuntu-latest
    needs: test
    steps:
    - uses: actions/checkout@v4
    - name: Build Debug APK
      run: ./gradlew assembleDebug
```

## Criterio de Done

- [ ] Workflow creado
- [ ] Tests se ejecutan en push/PR
- [ ] Artifacts disponibles
