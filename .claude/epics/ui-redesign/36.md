# Task: Crear Ilustraciones y Assets Minimalistas

## Descripción

Crear o sourcing de ilustraciones e iconos con estilo minimalista consistente para toda la app.

## Estilo Visual Definido

### Características
- **Líneas simples** - Stroke de 2dp
- **Formas geométricas** - Círculos, rectángulos redondeados
- **Colores de la paleta** - Primary, secondary, success
- **Sin caras** - Personajes abstractos si se necesitan
- **Tamaño moderado** - No dominar la pantalla

### Paleta para Ilustraciones
```
Primary: #6366F1 (Indigo)
Secondary: #8B5CF6 (Violeta)
Accent: #10B981 (Verde menta)
Neutral: #94A3B8 (Gris)
Background elements: #E2E8F0 (Gris muy claro)
```

```xml
<vector android:width="200dp" android:height="200dp">

</vector>
```

## Opciones de Sourcing

### Opción A: Crear desde cero
- Usar Figma o similar
- Exportar como vector XML
- Más tiempo pero 100% custom

### Opción B: Adaptar de librerías
- [unDraw](https://undraw.co/) - Ilustraciones customizables
- [Lucide](https://lucide.dev/) - Iconos
- [LottieFiles](https://lottiefiles.com/) - Animaciones
- Adaptar colores a nuestra paleta

### Opción C: Híbrido (Recomendado)
- Iconos: Material Symbols
- Ilustraciones simples: Crear en código (vector drawable)
- Animaciones: LottieFiles adaptadas
- Ilustraciones complejas: unDraw
