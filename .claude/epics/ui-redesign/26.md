# Task: Implementar Design System Foundation

## Descripción

Crear la base del sistema de diseño de Umbral con colores, tipografía, espaciado y tokens de diseño que se usarán en toda la app.

6. **Theme.kt** - Tema completo con light/dark
```kotlin
@Composable
fun UmbralTheme(
    darkTheme: Boolean = isSystemInDarkTheme(),
    content: @Composable () -> Unit
) {
    val colorScheme = if (darkTheme) DarkColorScheme else LightColorScheme

    MaterialTheme(
        colorScheme = colorScheme,
        typography = UmbralTypography,
        shapes = UmbralShapes,
        content = content
    )
}
```
